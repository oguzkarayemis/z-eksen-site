# z-eksen e-yayıncılık
Bu depo, z-eksen e-yayıncılık'ın IPFS uyumlu statik web sitesinin kaynak kodlarını barındırır. "Dijital Zanaatkârlık" ve "Anti-Moloch" felsefesiyle, sıfır JS (Astro) mimarisi kullanılarak inşa edilmiştir.

## Mimari Özellikler
- **Framework:** Astro (Static Site Generation)
- **Barındırma:** IPFS (InterPlanetary File System) & ENS
- **Veritabanı:** Markdown tabanlı İçerik Koleksiyonları (Zod Type Safety)
- **Stil:** Vanilla CSS (Dışa bağımlılık olmaksızın yerel fontlar)

## Geliştirici Kurulumu
1. Bağımlılıkları yükleyin: `npm install`
2. Yerel sunucuyu başlatın: `npm run dev`
3. IPFS için statik derleme: `npm run build`
